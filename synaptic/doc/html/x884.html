8. Keyboard Shortcuts

List of all global short cuts in Synaptic Package Manager:

Ctrl-E Execute all queued changes.
Ctrl-F Open the package find dialog.
Ctrl-G Queue all possible upgrades.
Ctrl-I Queue the selected package(s) for installation.
Ctrl-N Ignore any queued changes and set package as before.
Ctrl-F Open the package find dialog.
Ctrl-G Queue the whole system for upgrade.
Ctrl-I Queue the selected package(s) for installation.
Ctrl-N Ignore any queued changes and set package as before.
Ctrl-Q Quit Synaptic Package Manager.
Ctrl-R Queue the selected package(s) for removal.
Ctrl-U Update the list of known package.
