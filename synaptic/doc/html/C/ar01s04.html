Repositories

Packages are made available through so called repositories. The index of the repository provides basic information about all available packages, e.g. required dependencies or a short description. Synaptic Package Manager merges all index files to a single package list.

Repositories can be located on many medias: CD-Rom/DVD, local hard disk, internet (http, ftp) or remote file system (nfs).

To Refresh the Package List

To refresh and rebuild the list of known packages:

  • Choose Actions->Refresh Package List from the menu.

  • Click on Refresh List in the toolbar.

To Add, Remove or Edit Repositories

You can edit, add or remove repositories in the repository preferences:

Choose Preferences->Repositories from the menu.
