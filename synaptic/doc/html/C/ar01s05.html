Filters

Filters enable you to narrow down the list of shown packages.

Synaptic Package Manager comes with a set of predefined filters:

Table 4.  Predefined Filters

Filter Description
All packages Lists all known packages.
Install Lists only currently installed packages.
Not installed Lists only currently not installed packages.
Upgradable Lists only packages with a later version available.
Broken Lists only packages with broken dependencies.
Programmed Changes Lists only marked packages.
New in archive Lists only new packages since the last refresh.
Residual Lists only former installed packages that left data or configuration files on the system.
Pkg with Debconf Lists only packages that can be configured through the debian package configuration system (debconf).
Obsolete or locally installed Lists only packages that are not (for longer) included in one of the specified repositories.
Search Filter Lists only matching packages of the last package search, see To list only matching packages .

To Apply Filters

To apply a filter perform one of the following steps:

  • Choose the filter from the View->Apply Filter menu.

  • Choose the filter from the Show combobox in the main window.

To Edit, Remove or Add Filters

Filters can be added, removed and modified in the filter preferences:

  • Choose from the menu View-> Edit Filters....

  • Click on Filters... in the main window.
