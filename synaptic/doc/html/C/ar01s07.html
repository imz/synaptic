Keyboard Shortcuts

List of all global short cuts in Synaptic Package Manager:

Ctrl-R Refreshes the package list.
Ctrl-I Marks the selected package(s) for installation.
Delete Marks the selected package(s) for removal.
Shift-Delete Marks the selected package(s) for removal including configuration files.
Ctrl-U Marks the selected package(s) for upgrade.
Ctrl-G Marks all possible upgrades.
Ctrl-P Applies all marked changes.
Ctrl-N Unmarks package(s).
Ctrl-F Opens the package find dialog.
Ctrl-Q Quits Synaptic Package Manager.
